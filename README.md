# excel-desktop
今回作成したアプリは Excelを読み込むんでグラフを作成するアプリです。


左下のOpen Fileボタンを押すと Excelファイルを選ぶと、x軸とy軸の種類を選択できるようになり、Plot Graphボタンを選択するとグラフを描画できるアプリです。

この時下のLineとScatterでグラフを折線グラフか散布図かを選ぶ事ができるようになります。
